# ④-a T3 活性验证（v5.9.1）

对初步判定为 T3 的技能，执行活性验证后决定是否降级：

1. **使用证据检查**（复用 ②-ter #5）：有使用记录 → 降 T2 📋 有使用记录，保留
2. **近30天修改检查**：30天内有修改 → 降 T2 🕐 近期修改，保留
3. **用户定期触发检查**：触发词在 session 日志中出现过 → 降 T2 🔁 有触发记录，保留
4. 三项均无 → 保持 T3 📦 建议归档

**影响**：降级的 T2 不再计入 T3 比例，降低误归档率。

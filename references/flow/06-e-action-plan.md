# 下一步建议转介

本文件只定义报告中的转介方式，不定义执行阶段。

当翻译精炼、项目画像或推荐发现后续工作时，说明：

| 发现 | 建议直接调用 | 必要输入 |
|---|---|---|
| 需要写入用户自制技能 | 技能编辑 Agent | 目标路径、候选文本、修改确认 |
| 需要安装、更新或卸载 | 插件/技能安装 Agent | 候选来源、作用域、确认 |
| 需要发布 | 发布 Agent | 仓库、分支、发布范围 |
| 需要配置、桌面迁移或历史清理 | 对应环境维护 Agent | 目标环境、备份边界、确认 |

不得代替这些 Agent 执行操作，也不得生成执行命令。
